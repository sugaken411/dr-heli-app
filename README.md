# 🚁 AW109 EMS System (v13.0 / v14.0 管理)

航空医療・ドクターヘリ運用のための総合DXプラットフォーム。
現場でのオフライン入力、電子カルテへの転記、デブリーフィング、そして多職種での事案振り返りをシームレスに連携します。

## 🌟 v13.0 / v14.0 の主なアップデート内容
* **全画面共通アーキテクチャ標準の強制適用:** 現場の不安定な電波環境に完全対応するため、5秒で電波警告、15秒でタイムアウト・リロードを指示する「インテリジェントローダー」を全画面に必須搭載。
* **通信エラー監視の最適化:** iOS PWA特有の電波瞬断時に発生する無害な `Load failed` および `Failed to fetch` エラーを検知してGASへの自動送信から除外する高精度フィルターを標準化。
* **日付フォーマットのパディング正規化:** スプレッドシート上の `YYYY/M/D` 形式とHTML5側の `YYYY-MM-DD` 形式の表記揺れを吸収し、検索漏れ・カレンダーへの実績非表示バグを論理的に完全排除する `normalizeDate` ロジックを全検索・カレンダー連携に実装。
* **マスター管理CRUD機能の拡張:** 管理コンソール（admin.html）から、スプレッドシートを直接開くことなく「お知らせ」「取扱説明書」「Q&A」の完全な追加・修正・削除（CRUD）操作を可能とするAPIを実装。
* **多次元動的クロス分析（BI）への進化:** 統計画面（summary.html）にて、縦軸・横軸をユーザーが自由に選択して再計算できる「Pivotヒートマップ」および「動的カスタマイズレーダー」を導入。

## 📂 アプリケーション構成
1. `portal.html` - 全アプリの玄関口（未完了アラート・お知らせ・マニュアル表示）
2. `index.html` - 症例登録アプリ（現場入力用：移動用酸素・呼吸器の段組みUI、エコー・ベニューラ階層化対応）
3. `search.html` - 事案検索コンソール（カルテ完了処理・データ修正・CSV出力）
4. `management.html` - 統括・業務管理（月間カレンダー・出動状況および待機理由一覧）
5. `report.html` - 業務日報（浜町日誌・タイムライン自動生成・印刷・太陽位置連動計算）
6. `checklist.html` - 統合チェックリスト（日常点検入力・履歴および判定管理）
7. `debriefing.html` - デブリーフィング（多職種振り返り・疾患および病態評価入力）
8. `debrief_search.html` - 教育・事案検索（過去事例の学習用・クロス検索アーカイブ）
9. `library.html` - 資料集・アーカイブ（外部資料データベースへの接続）
10. `admin.html` - マスター管理コンソール（各種マスタ追加・Q&A回答・CRUD制御機能）

## ⚙️ 技術仕様・連携
* **バックエンド:** Google Apps Script (GAS) Webアプリケーション（単一doPost構成）
* **データベース:** Google Spreadsheet（1行目ヘッダー文字列完全一致による動的インデックスマッピング制御）
* **フロントエンド:** HTML5 / CSS3 (Tailwind CSS) / Vanilla JavaScript (外部ライブラリ：Chart.js, SunCalc)
* **PWA対応:** `manifest.json` と `sw.js` により、iPad/スマートフォンでのスタンドアロン起動と6時間強固キャッシュに対応。
